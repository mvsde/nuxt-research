# Nuxt Research

## Installation

[Node.js](https://nodejs.org)

```bash
# Dependencies
npm install
```

## Development & Production

```bash
# Start dev server
npm run dev

# Build production files
npm run build

# Start production server
npm run start

# Generate production files for static file server
npm run generate
```

## Linting

```bash
# Lint JavaScript
npm run lint:script

# Lint CSS
npm run lint:style
```
